About

My Journey

Saya memulai ketertarikan di bidang teknologi sejak kelas 6 SD dengan belajar pemrograman Python secara otodidak. Dari situ saya semakin tertarik pada backend development dan cybersecurity. Awalnya saya membuat tools spam sederhana dan sempat memperoleh penghasilan dari sana. Namun saya menyadari dampak negatifnya, sehingga beralih fokus ke pengembangan web dan membangun website pertama saya. Saya kemudian memperdalam backend development dengan mempelajari Node.js serta database seperti PostgreSQL dan MongoDB, yang akhirnya menjadi bidang yang paling saya minati.

Selama mendalami backend, saya memahami pentingnya aspek keamanan, sehingga mulai mempelajari cybersecurity, khususnya penetration testing, dan menekuni peran sebagai bug hunter. Hingga saat ini, saya telah menyelesaikan lebih dari 5 audit keamanan, menemukan serta melaporkan berbagai kerentanan pada platform dan produk, serta menerima beberapa sertifikasi apresiasi dari universitas dan instansi pemerintah.

Ketertarikan saya kemudian berkembang ke bidang Open Source Intelligence (OSINT), dengan fokus pada teknik reconnaissance dan information gathering. Melihat maraknya penipuan online, saya mengembangkan tools OSINT untuk membantu menelusuri jejak digital berbasis nomor telepon dari data breach. Saya juga bergabung dengan MataHacker.id sebagai mentor sekaligus pengembang tools pentesting dan OSINT.
